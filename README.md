# Courier Delivery Tracking System (Java)

## 📌 Project Description

The Courier Delivery Tracking System is a console-based Java application developed using Core Java and Object-Oriented Programming (OOP) concepts. It allows users to track the status of parcels using a unique parcel ID. The system simulates real-world courier services where users can check whether their parcel is dispatched, in transit, out for delivery, or delivered.

---

## 🎯 Objectives

* To design a parcel tracking system using Java classes
* To manage and update delivery status
* To implement tracking using unique IDs
* To simulate a real-world courier system
* To strengthen OOP and logical thinking skills

---

## ⚙️ Features

* Add new parcel
* Track parcel by ID
* Update parcel status
* Menu-driven console interface
* Uses ArrayList for storing parcels

---

## 🛠️ Technologies Used

* Java (Core Java)
* OOP Concepts (Class, Object, Encapsulation)
* ArrayList
* Scanner (for user input)

## 🚀 How to Run

1. Install Java JDK
2. Open the project in any IDE
3. Save file as `CourierSystem.java`
4. Compile the program:

   ```
   javac CourierSystem.java
   ```
5. Run the program:

   ```
   java CourierSystem
   ```
